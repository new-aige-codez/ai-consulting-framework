**Questions to Validate (Ask user if answers are not found in context):**

**Basic UX Structure Questions:**
- What are the main screens/pages needed?
- Navigation structure (header, sidebar, bottom nav)?
- Key buttons and their placement?
- Input fields and form structures?
- Dropdown menus and selection elements?
- Text hierarchy (headers, subtext, body)?

**User Experience Structure:**
- What are the main user journeys from entry to goal completion?
- Do you need user accounts, authentication, or can it be anonymous?
- Are there different user roles with different permissions?
- What are the core actions users need to perform?

**Data & Business Logic:**
- What data do users create, read, update, or delete?
- Are there any complex business rules or calculations?
- Do you need reporting, analytics, or data export features?
- How should data be organized and categorized?

**Integration & Advanced Features:**
- Do you need to integrate with external services (payment, email, etc.)?
- Are there any workflow or approval processes?
- Do you need notifications, alerts, or communication features?
- Are there any automation or scheduled task requirements?

**Platform & Device Considerations:**
- What are the target devices and primary screen sizes?
- Does the app need to function offline or with poor connectivity?
- Are there any accessibility requirements (WCAG compliance)?
- Do you need mobile apps, or is web sufficient?

**Optional Uploads:**
- Reference screenshots or wireframes
- Basic color preferences
- Style inspirations

**Note:** Focus is on functionality and basic layout. Advanced styling will be handled in separate workflow after MVP completion.
